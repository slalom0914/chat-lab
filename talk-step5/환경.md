### 환경 설정
1. npm init -y

2. npm i koa -> npm i express(더 범용)
:요청객체와 응답객체를 제공해준다. - rest api - GET, POST, PUT, DELETE
:http프로토콜 이용함. - 단방향 - 
:페이지 제공
웹서비스기반이다.

3. 클라이언트 서버 측 분리
서버 : src/main.js
클라이언트 : src/public/client.js
-> 하나의 프로젝트에서 둘 다  공존함.
